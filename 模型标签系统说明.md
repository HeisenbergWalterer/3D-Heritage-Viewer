# 模型标签和分类系统

## 📋 功能说明

模型标签系统为每个3D文物模型添加了**朝代**和**类别**标签，便于分类和识别。

## 🏷️ 标签类型

### 1. 朝代标签 (Dynasty)
显示文物所属的历史朝代，如：
- **清代** - 清朝时期的文物
- **唐代** - 唐朝时期的文物  
- **夏商** - 夏商时期的文物
- **未知** - 朝代不明确的文物

样式：橙色背景 + 棕色文字

### 2. 类别标签 (Category)
显示文物的类型分类，如：
- **陶瓷器** - 陶瓷类文物
- **雕塑** - 雕塑类文物
- **青铜器** - 青铜器类文物
- **乐器** - 乐器类文物
- **书画** - 书画类作品
- **玉器** - 玉器类文物
- **金银器** - 金银器类文物
- **织绣** - 纺织品类

样式：蓝色背景 + 深蓝色文字

## 📝 数据结构

在 `data/descriptions.json` 中，每个模型的数据结构如下：

```json
{
  "模型名称": {
    "title": "文物标题",
    "dynasty": "朝代名称",
    "category": "类别名称",
    "description": "详细描述文本"
  }
}
```

## 🎨 标签样式

### 朝代标签
- **背景色**: `#FFE5B4` (浅橙色)
- **文字颜色**: `#8B4513` (棕色)
- **边框颜色**: `#8B4513` (棕色)

### 类别标签
- **背景色**: `#E6F3FF` (浅蓝色)
- **文字颜色**: `#0066CC` (深蓝色)
- **边框颜色**: `#0066CC` (深蓝色)

样式定义在 `styles.css` 中：
```css
.tag-dynasty {
    background-color: #FFE5B4;
    color: #8B4513;
    border-color: #8B4513;
}

.tag-category {
    background-color: #E6F3FF;
    color: #0066CC;
    border-color: #0066CC;
}
```

## ➕ 添加新模型

### 步骤：

1. **准备模型文件**
   - 将 `.gltf` 或 `.glb` 文件放入 `assets/` 文件夹

2. **添加描述数据**
   - 编辑 `data/descriptions.json`
   - 添加新的模型条目：
   ```json
   {
     "新模型名称": {
       "title": "文物名称",
       "dynasty": "所属朝代",
       "category": "文物类别",
       "description": "详细描述..."
     }
   }
   ```

3. **更新模型选择器**（可选）
   - 编辑 `main.html`
   - 在模型列表中添加：
   ```html
   <div class="model-item" data-model="新模型名称.gltf">显示名称</div>
   ```

### 示例：添加一个书画作品
```json
{
  "painting": {
    "title": "清明上河图",
    "dynasty": "宋代",
    "category": "书画",
    "description": "《清明上河图》是北宋画家张择端的存世精品，描绘了北宋都城东京（今河南开封）的繁华景象..."
  }
}
```

## 🎯 扩展建议

### 添加更多标签类型

如果需要添加其他标签（如年代、材质、尺寸等），可以：

1. **更新 JSON 结构**
   ```json
   {
     "model": {
       "title": "文物名称",
       "dynasty": "清代",
       "category": "陶瓷器",
       "material": "青瓷",
       "year": "康熙年间",
       "size": "高15cm",
       "description": "..."
     }
   }
   ```

2. **更新 descriptionManager.js**
   ```javascript
   if (modelData.material) {
       htmlContent += `<span class="tag tag-material">${modelData.material}</span>`;
   }
   ```

3. **添加样式到 styles.css**
   ```css
   .tag-material {
       background-color: #E8F5E9;
       color: #2E7D32;
       border-color: #2E7D32;
   }
   ```
